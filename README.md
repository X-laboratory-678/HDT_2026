# HDT_2026

Data and code repository for the manuscript:

**Corridor optimization enables cost-competitive and low-emission battery electric long-haul road freight**

This repository accompanies the study on corridor optimization for battery electric long-haul road freight. It is intended to organize the materials associated with the analysis, including data resources, computational workflows, and supporting documentation.

## Scope
The repository is structured to cover:
- datasets and processed data relevant to the study
- scripts for analysis, modeling, and figure generation
- documentation describing repository structure and reproducibility
